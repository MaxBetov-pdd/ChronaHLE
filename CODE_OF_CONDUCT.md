# ChronaHLE code of conduct

Be respectful, patient and constructive. ChronaHLE is maintained by volunteers;
support and compatibility work are not guaranteed on any schedule.

Harassment, discrimination, threats and targeted abuse are not tolerated.
Project maintainers may remove content or restrict participation when needed to
keep the community safe and productive.

## Copyrighted software

Do not upload, link to, request or offer pirated applications, games, SDKs,
keys or extracted proprietary assets. "Abandonware" is not a legal exception.
Bug reports should contain metadata and logs, never the application binary.

See [CONTRIBUTING.md](CONTRIBUTING.md) for repository rules.
