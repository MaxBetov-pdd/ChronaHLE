# Fork notice

ChronaHLE is an independent project derived from the open-source touchHLE
project: <https://github.com/touchHLE/touchHLE>.

The fork was created from touchHLE commit
`0f2c163005d7511030be57b70396f4910bbc6d1a`. The original Git history is kept
in this repository and the original project is configured locally as the
`upstream` remote.

ChronaHLE is not an official touchHLE release and is not endorsed or supported
by the touchHLE maintainers. Bugs introduced by ChronaHLE should be reported to
ChronaHLE, not to the upstream project.

Source files derived from touchHLE remain under MPL-2.0 and retain their
license notices. Third-party submodules, fonts and runtime libraries keep their
respective licenses and copyright notices.

The ChronaHLE name, icon, compatibility additions, platform changes and other
new modifications are maintained by ChronaHLE contributors.
